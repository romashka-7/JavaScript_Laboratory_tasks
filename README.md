# Интерактивные задачи на HTML, CSS и JavaScript

Этот проект содержит задания, которые помогают закрепить базовые знания следующих технологий: HTML, CSS, JavaScript.

# Список заданий

1. Вывод времени и даты
   
   Отображение текущего времени (часы, минуты, секунды с am/pm), даты (день, месяц, год) и дня недели. 

2. Календарь

   Интерактивный календарь с возможностью переключения месяцев. Текущий день подсвечивается, выходные отмечены цветом.

3. Подсчёт элементов в DOM-дереве
   
   Подсчёт количества всех тегов `div` и `table` на странице.

4. Динамическая деформация HTML-элементов
   
   При клике на цветные блоки они увеличиваются, вращаются, а затем начинают менять цвет.

5. Создание `div` с помощью радиокнопок
    
   Выбор типа содержимого (текст или случайное изображение) → создание кликабельного блока → удаление блока по клику.

6. Работа со свойством CSS `:hover`
    
   При наведении на блок он смещается вправо, при уходе мыши — возвращается обратно.

7. Аккордеон и удаление элементов
    
   Раскрывающийся список сладостей (аккордеон). При клике на элемент списка он плавно исчезает и удаляется.

8. Работа с действиями и аудио
    
   Аудиоплеер (play/pause) и эффект наведения, скрывающий картинку и показывающий текст.

9-10. Форма с валидацией
    
   Поля: имя, фамилия, отчество, дата рождения, факультет, кафедра, email, логин, пароль и подтверждение пароля.  
   Регулярные выражения проверяют:
   - русские буквы с заглавной
   - дату
   - email
   - логин 
   - сложный пароль
   
   Поля подсвечиваются зелёным (верно) или красным (ошибка). Пароли проверяются на совпадение.

# Как открыть

1. Склонируйте репозиторий
2. Откройте `index.html` в браузере
3. Для работы `task5.js` и `task8.js` убедитесь, что папки `images/` и `audio/` находятся в корне проекта
